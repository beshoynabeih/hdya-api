# hdya-api

## how to install


```bash
  $ python -m venv .vevn
  $ source .venv/bin/active
  $ pip install -r requirments.txt
```

  Setup database credentials in settings.py

```bash
  $ python manage.py makemigrations
  $ python manage.py migrate
```
  Finally
```bash
  $ python manage.py runserver
```
